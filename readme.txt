Custom Customer Notes for WooCoommerce makes custom customer notes in WooCoomerce. 

== Description ==

This is a lightweight plugin for WooCommerce that makes it possible to instant add customer or private notes with predefined messages.
== Installation ==

Installing "Custom Customer Notes for WooCommerce" can be done either by searching for "Custom Customer Notes for WooCommerce" via the "Plugins > Add New" screen in your WordPress dashboard, or by using the following steps:

1. Download the plugin via WordPress.org
1. Upload the ZIP file through the 'Plugins > Add New > Upload' screen in your WordPress dashboard
1. Activate the plugin through the 'Plugins' menu in WordPress
